# Cancellation and Rescheduling Policy

Patients can cancel a confirmed appointment for free up to a doctor's
cancellation window before the appointment start time. Most doctors set this
to 2 hours, though some allow more; the minimum any doctor can set is 1 hour.
After that window passes, the cancel button is disabled in the app and the
appointment must be attended or it will be marked as a missed visit (no-show)
by the doctor.

Rescheduling works the same way: it cancels your current appointment (subject
to the same policy window) and opens a fresh booking at the new time, which
still needs to be confirmed like any new appointment — it is not
automatically confirmed just because it replaces an old one.

If a doctor cancels an appointment, there is no policy window — the patient
is notified immediately, offered the doctor's next available slots, and
automatically considered for any matching waitlist.
